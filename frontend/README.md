# 🔮 塔罗牌小程序

一款功能完善的智能塔罗牌占卜小程序，支持微信小程序和 H5。

## ✅ 功能特性

### 核心功能
- **78张完整塔罗牌库** - 22张大阿尔卡纳 + 56张小阿尔卡纳
- **6种经典牌阵**
  - 单张牌阵（答疑）
  - 三张牌阵（过去-现在-未来）
  - 关系牌阵（双人关系）
  - 二选一牌阵（决策）
  - 事业牌阵（职场发展）
  - 凯尔特十字（深度探索）

### AI 智能解读
- 支持多种 AI 提供商：OpenAI、Claude、阿里云、Moonshot Kimi
- 基于牌阵位置的个性化解读
- 结合求占问题的深度分析
- 本地快速解读（无需 API）

### 每日一抽
- 每天一次机会，基于日期与设备生成唯一牌面
- 自动计算月相阶段
- 按元素类型的个性化日运指引
- 历史日运记录

### 牌意详解
- 78张牌的完整详解
- 正位/逆位实时切换
- 感情、事业、成长三维度建议
- 元素、星体等占星信息

### 体验优化
- 洗牌动画效果
- 抽牌震动反馈
- 流畅的页面过渡
- 紫色神秘主题
- 星空背景动画

### 其他功能
- 历史占卜记录保存
- 分享功能（复制文本/图片）
- 本地数据持久化
- 响应式布局

## 🚀 技术栈

| 项目 | 技术 |
|-----|-----|
| 框架 | uni-app (Vue3) |
| 样式 | SCSS |

```
tarot-app/
├── components/           # 组件
├── data/

│   ├── tarot.js           # 塔罗核心逻辑
│   ├── ai.js              # AI 解读接口

│   ├── index/             # 首页 - 牌阵选择
```

## 🌐 运行

```bash
# 安装依赖
npm install

# 运行到微信小程序
npm run dev:mp-weixin

# 运行到 H5
npm run dev:h5

# 构建发布
npm run build:mp-weixin
```

## 📖 使用说明

### 1. 快速开始
- 打开小程序，选择牌阵
- 输入你的问题（可选）
- 点击"开始洗牌"
- 按顺序点击抽取牌
- 查看 AI 解读或基础解读

### 2. 配置 AI 解读
在结果页面点击"配置 API"：
- 选择 AI 提供商
- 输入 API Key
- 数据仅存储在本地

### 3. 每日一抽
- 切换到"日运" Tab
- 每天只能抽一次
- 基于日期确保每人每天同一牌面
- 查看个性化日运指引

## 🎨 牌面图片

```
static/images/cards/
├── major/
│   ├── 0.png   # 愚者
│   ├── 1.png   # 魔术师
│   └── ...     # 0-21
├── wands/
│   ├── 1.png   # 权杖首牌
│   └── ...     # 1-14
├── cups/
├── swords/
└── pentacles/
```

没有图片时会自动显示 SVG 占位图。
